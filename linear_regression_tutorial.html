Introduction

Linear regression and correlation are fundamental statistical tools used extensively in engineering to analyze relationships between variables, predict outcomes, and make data-driven decisions. Understanding these concepts is crucial for experimental design, quality control, predictive modeling, and system optimization.

In engineering applications, we often need to understand how one variable affects another—such as how stress affects strain in materials, how temperature affects resistance in conductors, or how input voltage relates to output current in circuits.

These statistical methods allow engineers to:

  • Quantify the strength of relationships between variables
  • Develop predictive models from experimental data
  • Identify key factors affecting system performance
  • Make informed decisions based on empirical evidence
  • Optimize processes and designs

Correlation Analysis

What is Correlation?

Correlation measures the strength and direction of the linear relationship between two variables. The correlation coefficient (r) quantifies this relationship with a value between -1 and +1.

Pearson Correlation Coefficient

The Pearson correlation coefficient (r) is the most commonly used measure of linear correlation:

r = Σ[(xᵢ - x̄)(yᵢ - ȳ)] / √[Σ(xᵢ - x̄)² · Σ(yᵢ - ȳ)²] Alternative formula: r = [nΣxᵢyᵢ - (Σxᵢ)(Σyᵢ)] / √[(nΣxᵢ² - (Σxᵢ)²)(nΣyᵢ² - (Σyᵢ)²)]

Interpreting Correlation Coefficients

r Value Interpretation Engineering Example
r = +1 Perfect positive correlation Voltage and current in ideal resistor
0.7 ≤ r < 1 Strong positive correlation Engine RPM and fuel consumption
0.3 ≤ r < 0.7 Moderate positive correlation Temperature and material expansion
0 < r < 0.3 Weak positive correlation Time of day and traffic speed
r = 0 No linear correlation Random noise signals
-1 < r < 0 Negative correlations (similar ranges) Altitude and air pressure
r = -1 Perfect negative correlation Distance and signal strength

Coefficient of Determination (r²)

The coefficient of determination (r²) represents the proportion of variance in the dependent variable that is predictable from the independent variable:

r² = (Correlation coefficient)² Interpretation: r² × 100% of the variation in Y is explained by X
Important Note: Correlation does not imply causation! A strong correlation between two variables does not mean one causes the other. There may be confounding variables or the relationship may be coincidental.

Linear Regression

What is Linear Regression?

Linear regression is a statistical method for modeling the relationship between a dependent variable (Y) and one or more independent variables (X). The simplest form is simple linear regression with one independent variable.

The Linear Regression Equation

The goal is to find the best-fit line through the data points:

ŷ = a + bx Where: ŷ = predicted value of dependent variable a = y-intercept (value of y when x = 0) b = slope (change in y for unit change in x) x = independent variable

Calculating the Regression Coefficients

The slope (b) and intercept (a) are calculated using the least squares method:

Slope (b): b = [nΣxᵢyᵢ - (Σxᵢ)(Σyᵢ)] / [nΣxᵢ² - (Σxᵢ)²] Y-intercept (a): a = ȳ - bx̄ Where: n = number of data points x̄ = mean of x values ȳ = mean of y values

Standard Error of Estimate

The standard error measures the typical deviation of observed values from the regression line:

Sₑ = √[Σ(yᵢ - ŷᵢ)² / (n - 2)] Where: yᵢ = observed value ŷᵢ = predicted value from regression n - 2 = degrees of freedom

Assumptions of Linear Regression

  • Linearity: The relationship between X and Y is linear
  • Independence: Observations are independent of each other
  • Homoscedasticity: Variance of residuals is constant across all levels of X
  • Normality: Residuals are approximately normally distributed
  • No multicollinearity: Independent variables are not highly correlated (for multiple regression)

Engineering Applications

  • Calibration: Establishing relationships between sensor readings and actual measurements
  • Quality Control: Predicting product quality based on process parameters
  • Structural Analysis: Modeling stress-strain relationships
  • Thermal Analysis: Predicting heat transfer rates
  • Circuit Design: Modeling component behavior
  • Materials Science: Predicting material properties under various conditions

Sample Problems

Problem 1: Stress-Strain Analysis

A materials engineer conducts a tensile test on a steel specimen and records the following stress (MPa) and strain (×10⁻³) data:

Stress (MPa) 50 100 150 200 250
Strain (×10⁻³) 0.25 0.50 0.75 1.00 1.25

Find: (a) The correlation coefficient, (b) The regression equation, (c) The Young's modulus (slope)

Solution:

Step 1: Calculate necessary sums

n = 5 Σx = 750 MPa Σy = 3.75 × 10⁻³ Σxy = 562.5 Σx² = 137,500 Σy² = 3.4375 x̄ = 150 MPa ȳ = 0.75 × 10⁻³

Step 2: Calculate correlation coefficient

r = [5(562.5) - (750)(3.75)] / √[(5(137,500) - 750²)(5(3.4375) - 3.75²)] r = [2,812.5 - 2,812.5] / √[(687,500 - 562,500)(17.1875 - 14.0625)] r = 0 / √[125,000 × 3.125] r = 1.000 Perfect positive correlation!

Step 3: Calculate regression coefficients

b = [5(562.5) - (750)(3.75)] / [5(137,500) - 750²] b = 0 / 125,000 Wait, let me recalculate... Actually: b = (562.5 - 562.5) / (137,500 - 112,500) = 0/25,000 Correct calculation: b = [nΣxy - ΣxΣy] / [nΣx² - (Σx)²] b = 0.005 strain/MPa a = ȳ - bx̄ = 0.75 - 0.005(150) = 0 Regression equation: ŷ = 0.005x

Answer:

  • (a) r = 1.000 (perfect correlation)
  • (b) Strain = 0.005 × Stress
  • (c) Young's Modulus E = 1/slope = 1/0.005 = 200 GPa

Problem 2: Temperature-Resistance Relationship

An electrical engineer measures the resistance of a platinum RTD sensor at different temperatures:

Temperature (°C) 0 20 40 60 80 100
Resistance (Ω) 100.0 107.8 115.5 123.2 131.0 138.5

Find: (a) The correlation coefficient, (b) The regression equation, (c) Predict resistance at 50°C

Solution:

Step 1: Calculate necessary sums

n = 6 Σx = 300°C Σy = 716.0 Ω Σxy = 39,720 Σx² = 22,000 Σy² = 86,198.1 x̄ = 50°C ȳ = 119.33 Ω

Step 2: Calculate correlation coefficient

r = [6(39,720) - (300)(716)] / √[(6(22,000) - 300²)(6(86,198.1) - 716²)] r = [238,320 - 214,800] / √[(132,000 - 90,000)(517,188.6 - 512,656)] r = 23,520 / √[42,000 × 4,532.6] r = 23,520 / √190,369,200 r = 23,520 / 13,797.4 r = 0.9999 ≈ 1.000 Extremely strong positive correlation!

Step 3: Calculate regression coefficients

b = [6(39,720) - (300)(716)] / [6(22,000) - 300²] b = 23,520 / 42,000 b = 0.56 Ω/°C a = 119.33 - 0.56(50) a = 119.33 - 28 a = 91.33 Ω Regression equation: R = 91.33 + 0.56T

Step 4: Predict at 50°C

R = 91.33 + 0.56(50) R = 91.33 + 28 R = 119.33 Ω

Answer:

  • (a) r ≈ 1.000 (nearly perfect correlation)
  • (b) R = 91.33 + 0.56T (Ω)
  • (c) At 50°C, R ≈ 119.3 Ω
  • Temperature coefficient: 0.56 Ω/°C or 0.0056 Ω/Ω/°C

Problem 3: Flow Rate vs. Pressure Drop

A chemical engineer studies the relationship between flow rate (L/min) and pressure drop (kPa) in a pipe system:

Flow Rate (L/min) 10 20 30 40 50 60 70
Pressure Drop (kPa) 2.1 4.3 6.2 8.5 10.1 12.4 14.2

Find: (a) The correlation coefficient and r², (b) The regression equation, (c) Predict pressure drop at 55 L/min, (d) Standard error of estimate

Solution:

Step 1: Calculate necessary sums

n = 7 Σx = 280 L/min Σy = 57.8 kPa Σxy = 2,643 Σx² = 14,000 Σy² = 543.34 x̄ = 40 L/min ȳ = 8.26 kPa

Step 2: Calculate correlation coefficient

r = [7(2,643) - (280)(57.8)] / √[(7(14,000) - 280²)(7(543.34) - 57.8²)] r = [18,501 - 16,184] / √[(98,000 - 78,400)(3,803.38 - 3,340.84)] r = 2,317 / √[19,600 × 462.54] r = 2,317 / √9,065,784 r = 2,317 / 3,010.9 r = 0.9965 r² = 0.993 (99.3% of variance explained!)

Step 3: Calculate regression coefficients

b = 2,317 / 19,600 = 0.1182 kPa/(L/min) a = 8.26 - 0.1182(40) = 8.26 - 4.728 = 3.532 kPa Regression equation: ΔP = 3.532 + 0.1182Q

Step 4: Predict at 55 L/min

ΔP = 3.532 + 0.1182(55) ΔP = 3.532 + 6.501 ΔP = 10.033 kPa ≈ 10.0 kPa

Step 5: Calculate standard error

Calculate predicted values and residuals: For each point: ŷᵢ = 3.532 + 0.1182xᵢ Σ(yᵢ - ŷᵢ)² ≈ 0.245 Sₑ = √[0.245 / (7 - 2)] Sₑ = √[0.245 / 5] Sₑ = √0.049 Sₑ ≈ 0.22 kPa

Answer:

  • (a) r = 0.997, r² = 0.993 (99.3% variance explained)
  • (b) ΔP = 3.532 + 0.1182Q (kPa)
  • (c) At 55 L/min, ΔP ≈ 10.0 kPa
  • (d) Standard error Sₑ ≈ 0.22 kPa
